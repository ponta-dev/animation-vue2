## ディレクトリ構造
```
doc
  ドキュメント類
public
  基本のHtmlファイル、及びアイコン
  Vueコンポーネントが参照する画像類は別に保存する
src
  app
  common
  plugins
  App.vue
  main.ts
tests
.browserslistrc
  利用を前提とするブラウザのバージョン等を設定するファイル  
.env
  アプリケーション内で利用する環境変数を定義する
.eslintrc.js
  ESLINTの設定を定義する
.gitignore
  Gitのバージョン管理除外対象を定義する
babel.config.js
  babelの設定を定義する
cypress.json
  cypress(E2Eテストフレーム)の設定を定義する
jest.config.js
  jest(Unitテストフレーム)の設定を定義する
package-lock.json
  モジュール管理のロック定義
  直接編集はしない
package.json
  npmのモジュール管理の設定ファイル
README.md
  READMEファイル
tsconfig.json
  TypeScriptの設定を定義する
vue.config.json
  Vueに関する設定を定義する
  サーバーのプロキシ設定等を定義する
```
